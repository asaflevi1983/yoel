# 🚀 Space Explorer - יואל

A space-themed web app for Yoel — featuring NASA's Astronomy Picture of the Day and a space shooter game.

## Features

### 🌌 תמונת היום — Daily Space Image
- Fetches NASA's Astronomy Picture of the Day
- Browse previous/next days
- Image zoom, video support
- Cached in localStorage

### 🎮 משחק חלל — Space Shooter Game
- Fly a spaceship, destroy enemies, collect power-ups
- 5 enemy types including bosses every 5 levels
- Power-ups: ⚡ Rapid Fire, 🔱 Spread Shot, 🛡️ Shield, ❤️ Extra Life
- Increasing difficulty, high score tracking
- Keyboard and touch controls

## How to Play
- **Move**: Arrow keys / WASD / touch d-pad
- **Shoot**: Space bar / fire button / tap canvas
- **Navigate APOD**: Arrow buttons or ← → keys

## Tech
Pure HTML/CSS/JavaScript — no frameworks, no build tools. Runs on GitHub Pages.
